# Base Signal Board

## App Name

Base Signal Board

## Tagline

Broadcast directional conviction

## Description

Broadcast a directional signal on Base with title, intensity, and a short market-style explanation.

## Category

Market signal / directional board / onchain utility

## Short Pitch

Base Signal Board gives traders, builders, and observers a fast way to publish a directional call on Base with intensity, context, and a timestamped archive.

## Production Environment Variables

- `NEXT_PUBLIC_SIGNAL_BOARD_CONTRACT_ADDRESS`: deployed signal board contract address
- `NEXT_PUBLIC_BUILDER_CODE`: Base Builder Code after the app is registered
- `NEXT_PUBLIC_BASE_APP_ID`: Base.dev app id for domain ownership verification
